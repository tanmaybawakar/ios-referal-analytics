# iOS Referral Analytics

A simple, real-time analytics dashboard for ExpendoX referral sources.

## Features
- Real-time data from Supabase
- Interactive charts with Chart.js
- Date range filtering
- Responsive design
- Automatic refresh every 60 seconds

## Files
- `index.html`: Main dashboard with full features and error handling.
- `index-simple.html`: Simplified version of the dashboard.
